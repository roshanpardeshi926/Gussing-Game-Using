# 🎯 Guessing Game (Java)

A simple console-based number guessing game developed using Java. The program generates a random number, and the user tries to guess it with the help of hints.

---

## 🚀 Features

- 🔢 Random number generation
- ⌨️ User input using Scanner
- 📉 Hints (Too High / Too Low)
- 🔁 Multiple attempts until correct guess
- 🎉 Winning message on correct guess

---

## 🛠️ Technologies Used

- Java
- OOP Concepts (optional)
